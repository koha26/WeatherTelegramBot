# WeatherTelegramBot
Инструкция:
1) Открыть папку WeatherTelegramBot через любую IDE
2) Использовть Maven для загружки требуемых библиотек
3) Запустить файл Main.java

ИЛИ запустить файл WeatherTelegramBot\target\WeatherTelegramBot-1.0-SNAPSHOT-shaded.jar

4) Зайти в Telegram
5) Зайди в диалог с ботом по имени KohaBot
6) Набрать в поле ввода /start
